ajax请求数据方式：

常用：get/post
安全性
传输大小

传post要注意格式

其他请求方式 区别：
header  options    put    delete

restful 风格：
    后端提供四个接口：put(更新)/delete(删除)/get(获取)/post(提交)



get方式的数据：
        是添加到网址问号后面key=value
        多组数据通过：&符号链接username=xiaoming&password=123456
        安全性没有post好，查询数据
        网址长度有限制，get提交数据是有大小限制的。
        存成书签的时候，会保留数据。
        get返回结果，一般会有缓存。

post方式的数据：
        在一个隐藏体中，http协议传输过去
        安全性比get好，添加、删除、更新数据。
        post数据可以修改限制。
        存成书签的时候，不会保留数据。
        post一般不会有缓存问题。

post修改限制：



数据格式：key=value（键值对组合）

mysql基础：

1、MYSQL数据库通过sql语句进行数据库操作。    
   
2、创建数据库、表、字段    
        CREATE  DATABASE  数据库名称    
        CREATE  TABLE  表名称 ( 列名称1 数据类型,  列名称2 数据类型, ....... )     
  
3、数据的增删改查操作    
        查询         select * from 表 where 字段 = 值    
        添加         insert into 表(字段) values(值)    
        删除         delete from 表 where 字段 = 值   
        更新         update 表 set 字段 = 新值 where id = $id
