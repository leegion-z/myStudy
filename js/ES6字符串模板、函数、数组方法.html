箭头函数：
    写法简单 
    语义清晰 
    this绑定父级作用域（有的网站解释：箭头函数中没有this，她的this就是父级作用域的this）

    箭头函数注意事项：
        this 指向问题
        不能用于构造函数
        不可以使用arguments，使用时报错

var a = [1,2,3];
var b = a.map(function(item){   //.map映射 为返回值
    return item*item;      // ==> b=[1,4,9]
})

创建多个 li 
var a = [1,2,3];
var b = a.map(function(item){   
    return `<li>${item}</li>`;      
})

箭头函数 简写：
var a = [1,2,3];
var b = a.map ( item =>  `<li>${item}</li>` ) //一个参数 一个return 的简写
