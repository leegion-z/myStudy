2.4 webpack反向代理/公用模块/起@别名/css作用域


1、数据请求：

反向代理
        proxy:{
           '/v4/api/*': {
                  target: 'https://m.maizuo.com';,
                  host: 'm.maizuo.com',
                  changeOrigin:true
              }
        }
        /*
          在前端 发起 /v4/api/billboard/home请求
                转发到  http://m.maizuo.com/v4/api/billboard/home
         */





2、import 公用模块：
公用模块    
 
  new webpack.ProvidePlugin({
        axios: 'axios'  //各个模块都可以使用axios
   }),




3、起一个别名，方便访问
    
resolve: {
    alias: {
        '@assets': path.resolve("assets")
    }
        //起一个别名@assets
},



4.  css 局部作用域，已经集成在css-loader里了 ,加上?modules   ( css module 解决方案)  重启服务器
    loader: 'style-loader!css-loader?modules'
    loader: 'style-loader!css-loader?modules!sass-loader'

    (1)会产生一个独一无二的class (或者是id) 的名字，
    写css文件

    }

import styles  from "./index.scss";

......
